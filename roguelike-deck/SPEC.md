# カード式ローグライト 基幹システム仕様書 v0.6

## 1. 目的

本仕様書は、1人用カード式ローグライトゲームの根幹システムを定義する。

本作は、カードバトル型ローグライトをベースにしつつ、**ラン開始前にプレイヤー自身が作成した「オリジナルカード」を1枚持ち込める**ことを最大の特徴とする。

本作には複数キャラクターの概念は存在しない。
代わりに、ラン開始時に選択する**開始デッキ**と、その開始デッキに紐づく**初期遺物**によって、ランごとの基本戦術を変化させる。

また、敵図鑑を進めることで敵由来の強化パーツである**エネミーオーブ**を入手できる。
エネミーオーブは、オリジナルカードの**エネミースロット**に装着できる。

プレイヤーは、

* 開始デッキ
* 初期遺物
* オリジナルカード
* エネミーオーブ
* ラン中に獲得するカードや遺物
* ポーション
* カード進化
* 鍛冶によるカード強化
* 鍛冶による遺物変換
* 実績・称号
* 進行度解放

を組み合わせて、最終ボス撃破を目指す。

## 2. ゲームコンセプト

### 2.1 ジャンル

カードバトル型ローグライト。

---

### 2.2 コアコンセプト

**2枚のカードを合成して作った自分だけの切り札を軸に、開始デッキ・敵図鑑・進化カード・鍛冶を組み合わせて戦うカードローグライト。**

一般的なカードローグライトでは、キャラクターやクラスによって初期デッキやカードプールが変化する。
本作ではキャラクターを1人に固定し、代わりに以下の要素でランの個性を作る。

* 開始デッキ
* 初期遺物
* オリジナルカード
* エネミーオーブ
* ラン中の報酬選択
* カード進化
* 鍛冶

## 3. 基本体験

プレイヤーはラン開始前に、既存カード2枚を合成してオリジナルカードを作成する。
作成したオリジナルカードは、初期デッキに1枚だけ持ち込める。

オリジナルカード作成時に、特殊効果を直接付与することはできない。
代わりに、オリジナルカードには空のエネミースロットが存在する。

敵図鑑を進めることで、敵に由来するエネミーオーブを入手できる。
入手したエネミーオーブは、ラン開始前にオリジナルカードへ装着できる。

ラン開始時には、4種類程度の開始デッキから1つを選ぶ。
開始デッキにはそれぞれ初期遺物が設定されており、ランの基本的な戦術方針を決定する。

ラン中は、通常カード・進化カード・遺物・イベント・通常強化・鍛冶によってデッキを成長させ、最終ボス撃破を目指す。

本作の面白さは以下にある。

* 自分で作ったカードを活かす楽しさ
* 敵図鑑を進めて、敵由来の力を得る長期目標
* 素材カードの組み合わせを考える構築性
* 開始デッキごとの異なる立ち回り
* 初期遺物による戦術の方向づけ
* 進化カードをラン中に育てる楽しさ
* ポーションを使うか温存するかの短期判断
* 強力なカードを作る代わりに代償を背負うリスク管理
* ラン中に拾うカードや遺物で、最初の方針から変化していく偶然性
* スロット付きカードを鍛冶で育てる判断
* 不要な遺物を鍛冶で変換する判断
* 実績・称号・進行度解放による長期目標
* 高難度の試練レベルを攻略するやりこみ
* 「作成 → 挑戦 → 敵図鑑進行 → 解放 → 強化 → 再挑戦」のサイクル

## 6. ラン開始仕様

### 6.1 ラン開始時の選択

ラン開始時、プレイヤーは以下を選択する。

1. 開始デッキ
2. 持ち込みオリジナルカード
3. 装着するエネミーオーブ
4. 試練レベル

キャラクター選択は存在しない。

---

### 6.2 初期状態

ラン開始時、以下を設定する。

| 項目      | 初期値 |
| ------- | --: |
| 最大HP    | 100 |
| 現在HP    | 100 |
| 初期エナジー  |   3 |
| 初期ゴールド  | 100 |
| 基本ドロー   |   5 |
| 初期デッキ枚数 |  10 |
| 初期遺物    |   1 |

初期デッキ枚数は、開始デッキ9枚 + オリジナルカード1枚を基本とする。
オリジナルカードを持ち込まない場合は、代わりに「攻撃」を1枚追加して10枚にする。

開始デッキ固有カードの内容は、選択した開始デッキによって変化する。

## 7. 開始デッキシステム

### 7.1 概要

本作にはキャラクター選択は存在しない。

代わりに、ラン開始時に「開始デッキ」を選択する。

開始デッキは、初期カード構成と初期遺物をセットにしたものであり、キャラクターの代わりにランの基本方針を決める役割を持つ。

```text
開始デッキ
＋
初期遺物
＋
オリジナルカード
＋
エネミーオーブ
＝
ラン開始時の戦略
```

---

### 7.2 基本ルール

* ラン開始前に、開始デッキを1つ選択する。
* 開始デッキは4種類程度用意する。
* 各開始デッキには、専用の初期遺物が1つ設定される。
* 開始デッキはキャラクターではなく、戦闘スタイルとして扱う。
* プレイヤーキャラクター自体は1人のみとする。
* 開始デッキごとに、得意な戦術と苦手な戦術を持たせる。
* 開始デッキは、カード報酬の傾向に軽い影響を与える。
* 報酬補正の内部相性タグはプレイヤーには表示しない。

---

### 7.3 開始デッキ案

| 開始デッキ | 得意          | 苦手          | 初期遺物案                     |
| ----- | ----------- | ----------- | ------------------------- |
| 均衡型   | 攻撃、防御、安定    | 爆発力、特殊コンボ   | 戦闘開始時、少量のブロックを得る          |
| 連撃型   | 低コスト、ドロー、手数 | 高防御敵、妨害     | 1ターンに3枚目のカードを使うと小ダメージ     |
| 守護型   | ブロック、反撃、長期戦 | 火力不足、ブロック貫通 | ターン終了時にブロックが残っていれば小ダメージ   |
| 侵蝕型   | 毒、裂傷、弱体     | 短期決戦、状態耐性   | 1ターンに初めて状態異常を付与した時、その数値+1 |

---

### 7.4 開始デッキ詳細

#### 均衡型

攻撃と防御のバランスがよい標準型。
初心者向けで、どのオリジナルカードとも合わせやすい。

| 項目            | 内容               |
| ------------- | ---------------- |
| 得意            | 安定した攻防           |
| 苦手            | 爆発的なコンボ          |
| 初期遺物          | 古びた紋章            |
| 初期遺物効果        | 戦闘開始時、少量のブロックを得る |
| 向いているオリジナルカード | 攻撃 + 防御、攻撃 + ドロー |

---

#### 連撃型

軽いカードを連続で使い、手数で攻める型。

| 項目            | 内容                           |
| ------------- | ---------------------------- |
| 得意            | 低コスト、ドロー、複数回行動               |
| 苦手            | 高防御敵、手札妨害                    |
| 初期遺物          | 小さな歯車                        |
| 初期遺物効果        | 1ターンに3枚目のカードを使用した時、敵1体に小ダメージ |
| 向いているオリジナルカード | 攻撃 + ドロー、軽量化系、低コスト補助         |

---

#### 守護型

ブロックを維持しながら、反撃や長期戦で勝つ型。

| 項目            | 内容                           |
| ------------- | ---------------------------- |
| 得意            | 防御、反撃、長期戦                    |
| 苦手            | 火力不足、ブロック貫通、毒                |
| 初期遺物          | ひび割れた盾                       |
| 初期遺物効果        | ターン終了時にブロックが残っていれば、敵1体に小ダメージ |
| 向いているオリジナルカード | 防御 + 回復、防御 + 攻撃、防御 + 状態異常    |

---

#### 侵蝕型

毒・裂傷・弱体などの状態異常で敵を削る型。

| 項目            | 内容                           |
| ------------- | ---------------------------- |
| 得意            | 継続ダメージ、弱体、長期戦                |
| 苦手            | 短期決戦、状態耐性、複数敵の即処理            |
| 初期遺物          | 黒い小瓶                         |
| 初期遺物効果        | 1ターンに初めて状態異常を付与した時、その数値+1    |
| 向いているオリジナルカード | 状態異常 + 攻撃、状態異常 + 全体攻撃、毒 + 防御 |

### 7.5 開始デッキとオリジナルカード

開始デッキとオリジナルカードは、必ずしも同じ方向性である必要はない。

開始デッキはランの土台、オリジナルカードはランの主役として扱う。

| 開始デッキ | オリジナルカード | 戦略                |
| ----- | -------- | ----------------- |
| 連撃型   | 攻撃 + ドロー | 手数を増やして攻める        |
| 連撃型   | 防御 + 回復  | 連撃型の耐久不足を補う       |
| 守護型   | 高火力攻撃    | 守りながら大技を狙う        |
| 侵蝕型   | 全体攻撃 + 毒 | 複数敵に状態異常をばらまく     |
| 均衡型   | 代償付きカード  | 安定した初期デッキでリスクを支える |

## 8. オリジナルカードシステム

### 8.1 概要

オリジナルカードは、ラン開始前にプレイヤーが作成し、初期デッキに1枚だけ持ち込める特別なカードである。

本作では、オリジナルカードを完全自由入力で作るのではなく、**既存のカード2枚を合成する**ことで作成する。

```text
素材カードA
＋
素材カードB
＋
空のエネミースロット
＝
オリジナルカード
```

オリジナルカード作成時に、特殊効果を直接付与することはできない。

特殊効果は、敵図鑑の報酬として入手したエネミーオーブを、エネミースロットに装着することで付与する。

---

### 8.2 オリジナルカードの役割

オリジナルカードは、単なる強力な持ち込みカードではなく、そのランの戦略方針を決める「切り札」として扱う。

プレイヤーは、作成したオリジナルカードを軸にして、ラン中のカード報酬・遺物・強化方針を選択する。

| 合成内容              | 戦略の方向性      |
| ----------------- | ----------- |
| 攻撃カード + 防御カード     | 攻防一体の安定型    |
| 攻撃カード + ドローカード    | 手数で攻めるコンボ型  |
| 防御カード + 回復カード     | 長期戦向けの耐久型   |
| 状態異常カード + 全体攻撃カード | 複数敵に強い状態異常型 |
| 高コスト攻撃カード + 弱体カード | ボス戦向けの高火力型  |

---

### 8.3 作成手順

オリジナルカード作成は、ラン開始前の「カード工房」で行う。

```text
素材カードAを選択
 ↓
素材カードBを選択
 ↓
合成後の効果を確認
 ↓
完成コストを確認
 ↓
コスト圧縮の有無を選択
 ↓
代償を確認
 ↓
禁止組み合わせチェック
 ↓
カード名を設定
 ↓
空のエネミースロットを確認
 ↓
保存
 ↓
必要に応じてエネミーオーブを装着
 ↓
ランに持ち込む
```

---

### 8.4 基本ルール

* 素材カードは2枚まで選択できる。
* 完成カードは、素材カード2枚の効果を順番に持つ。
* 完成カードのコストは、素材カード2枚のコスト合計を基本とする。
* オリジナルカード作成時に特殊効果は付与できない。
* オリジナルカードには、空のエネミースロットを1つ持たせる。
* エネミースロットには、入手済みのエネミーオーブを1つだけ装着できる。
* 1回のランに持ち込めるオリジナルカードは1枚のみ。
* オリジナルカードは複製できない。
* ラン中にオリジナルカードを作り直すことはできない。
* ラン中にエネミーオーブを付け替えることはできない。
* オリジナルカードは通常のカード報酬やショップには出現しない。

---
### 8.5 コスト計算

完成したオリジナルカードの基本コストは、素材カード2枚のコスト合計とする。

```text
完成カードのコスト = 素材カードAのコスト + 素材カードBのコスト
```

| 素材カードA | 素材カードB | 完成コスト |
| ------ | -----: | ----: |
| 1コスト攻撃 | 1コスト防御 |     2 |
| 0コスト補助 | 1コスト攻撃 |     1 |
| 2コスト攻撃 | 1コスト弱体 |     3 |

---

### 8.6 コスト上限と圧縮

完成カードの基本コスト上限は3とする。

素材カードの合計コストが4以上になる場合、以下を適用する。

| 処理     | 内容                  |
| ------ | ------------------- |
| コスト圧縮  | コストを3に下げる代わりに代償を付ける |

コスト圧縮の例：

| 元の合計コスト | 完成コスト | 追加される代償例          |
| ------: | ----: | ----------------- |
|       4 |     3 | 使用後除外             |
|       4 |     3 | HPを失う             |
|       5 |     3 | 使用後除外 + 次ターンドロー減少 |
|       5 |     3 | 呪いカードを捨て札に加える     |

---

### 8.7 エネミースロット

エネミースロットは、オリジナルカード専用の特殊スロットである。

通常カードの特殊効果スロットとは異なり、鍛冶では埋められない。

エネミースロットには、敵図鑑の報酬で入手したエネミーオーブのみを装着できる。

| 状態   | 内容               |
| ---- | ---------------- |
| 空き   | エネミーオーブが装着されていない |
| 装着済み | エネミーオーブが装着されている  |
| ロック中 | ラン中のため変更できない     |

基本ルール：

* オリジナルカードは、作成時に空のエネミースロットを1つ持つ。
* エネミースロットは、オリジナルカード作成時には埋められない。
* エネミースロットには、エネミーオーブを1つだけ装着できる。
* エネミーオーブの付け替えは、ラン開始前のみ可能とする。
* ラン中にエネミーオーブを付け替えることはできない。
* MVPでは、オリジナルカード1枚につきエネミースロットは最大1つまでとする。

---

### 8.8 エネミーオーブ

エネミーオーブは、敵図鑑の進行報酬で入手できる特殊な強化パーツである。

各エネミーオーブは、特定の敵に由来する効果を持つ。

エネミーオーブをオリジナルカードのエネミースロットに装着することで、その敵に関連した特殊効果を得られる。

基本ルール：

* エネミーオーブは敵ごとに存在する。
* 敵図鑑のポイントが満タンになると、その敵に対応したエネミーオーブを入手できる。
* エネミーオーブはオリジナルカードに装着できる。
* エネミーオーブの効果は、その敵の特徴を反映したものにする。
* エネミーオーブは、通常カードには装着できない。
* エネミーオーブは、鍛冶では作成できない。
* エネミーオーブは、カード報酬やショップには通常出現しない。

---

### 8.9 エネミーオーブ例

| 敵タイプ   | オーブ効果例              |
| ------ | ------------------- |
| 毒を使う敵  | ダメージ後、毒を少量付与する      |
| 防御する敵  | 使用後、少量のブロックを得る      |
| 反撃する敵  | ブロックを得た後、敵に小ダメージ    |
| 召喚する敵  | 敵を倒した時、カードを1枚引く     |
| 弱体を使う敵 | ダメージ後、裂傷を付与する       |
| 高火力の敵  | 使用後除外される代わりに効果量を上げる |
| 回復する敵  | 使用後、HPを少量回復する       |
| 妨害する敵  | 使用後、次に引くカードを1枚確認する  |

---

### 8.10 エネミーオーブの制限

避けるべき効果：

* 敵の即死
* ボス無効化
* 無限ドロー
* 無限エナジー
* オリジナルカードの複製
* 戦闘開始時の大ダメージ
* プレイヤーが判断しなくても勝てる自動処理

エネミーオーブは、その敵の能力をそのままコピーするのではなく、カード効果として扱いやすい形に変換する。

---

### 8.11 複製不可ルール

オリジナルカードは、ゲームバランスを守るために複製不可とする。

以下の手段では、オリジナルカードを増やすことはできない。

* カード報酬
* ショップ
* イベント
* 遺物効果
* カード効果
* 戦闘中のコピー効果
* デッキ複製効果
* 一時生成効果

オリジナルカードは常にデッキ内に1枚のみ存在する。
MVP段階では、完全に複製不可とする。

---

### 8.12 禁止組み合わせ

以下の組み合わせは、原則として作成不可とする。

* 使用後に自身を生成する効果
* 自身を複製する効果
* 敵を即死させる効果
* ボスを無効化する効果
* 戦闘開始時に自動で大ダメージを与える効果
* プレイヤーが判断しなくても勝てる自動処理系効果

禁止組み合わせは、カード作成時に自動でチェックされる。

---

### 8.13 代償

強力な合成結果やコスト圧縮を行ったカードには、代償が付与される。

代償は単なるペナルティではなく、デッキ構築の方向性を生む要素として扱う。

| 代償     | 内容               |
| ------ | ---------------- |
| 使用後除外  | その戦闘中、再使用できない    |
| HP消費   | 使用時にHPを失う        |
| 手札破棄   | 使用時に手札を1枚捨てる     |
| 次ターン弱体 | 次ターンのドローやエナジーが減る |
| 山札汚染   | 捨て札に妨害カードを加える    |
| 敵強化    | 敵にブロックや攻撃上昇を与える  |
| ランダム対象 | 対象を選べない          |
| 使用条件   | 特定条件を満たさないと使えない  |

---

### 8.14 素材カードの制限

すべてのカードを素材にできるわけではない。

| カード種別    | 素材化               |
| -------- | ----------------- |
| 基本カード    | 可能                |
| 通常カード    | 可能                |
| 進化カード    | 原則不可、または進化前状態のみ可能 |
| 希少カード    | 一部可能              |
| 伝説カード    | 原則不可、または強い代償付き    |
| 呪いカード    | 原則不可              |
| 状態カード    | 原則不可              |
| オリジナルカード | 素材化不可             |

最初のMVPでは、基本カードと一部の通常カードのみ素材化可能とする。

---
### 8.15 オリジナルカードの保存

| 項目        | 内容             |
| --------- | -------------- |
| 保存数       | 初期は最大5枚程度     |
| 編集        | ラン開始前のみ可能      |
| ラン中編集     | 不可             |
| 持ち込み枚数    | 1枚             |
| 複製        | 不可             |
| 削除        | 可能             |
| 名前変更      | 可能             |
| 効果変更      | 素材カードの再選択として扱う |
| エネミーオーブ変更 | ラン開始前のみ可能      |

---

### 8.16 オリジナルカードの成長

オリジナルカードは、ラン中に通常強化の対象にできる。

ただし、通常カードより強くなりやすいため、強化内容には制限を設ける。

#### 通常強化

休憩所で、オリジナルカードを1回だけ通常強化できる。

| 元効果    | 強化後    |
| ------ | ------ |
| 6ダメージ  | 9ダメージ  |
| 5ブロック  | 8ブロック  |
| 毒2     | 毒3     |
| HPを3失う | HPを2失う |
### 8.17 ラン終了後の調整

ラン終了後、プレイヤーはオリジナルカードを調整できる。

調整では、以下を変更できる。

* 素材カードA
* 素材カードB
* カード名
* コスト圧縮の有無
* 代償の種類
* 装着するエネミーオーブ

### 8.18 ゲームとしての狙い

オリジナルカード合成システムの狙いは、以下である。

* プレイヤーに「自分で作ったカード」という愛着を持たせる。
* 完全自由生成によるバランス崩壊を防ぐ。
* 素材カードの組み合わせによって自然に戦略を生む。
* 敵図鑑とエネミーオーブによって長期的な強化目標を作る。
* 敵を知ることが、自分のカードを強くすることにつながる。

本作の独自性は、
**2枚のカードを合成し、敵由来の力を宿した自分だけの切り札を作ること**にある。

## 9. 敵図鑑システム

### 9.1 概要

敵図鑑は、登場した敵の情報を記録する恒久的なコレクション要素である。

敵ごとに図鑑ポイントが設定されており、ポイントが満タンになると、その敵に対応したエネミーオーブを入手できる。

敵図鑑は、やりこみ要素であると同時に、オリジナルカード強化の解放ルートでもある。

---

### 9.2 敵図鑑で表示する情報

| 項目       | 内容           |
| -------- | ------------ |
| 敵名       | 敵の名前         |
| 外見       | 敵の画像またはアイコン  |
| 出現階層     | 主に出現する階層     |
| HP目安     | おおよそのHP      |
| 行動パターン   | 確認済みの行動      |
| 使用する状態異常 | 毒、裂傷、弱体など    |
| 図鑑ポイント   | その敵の進行度      |
| 解放済み報酬   | 入手済みのエネミーオーブ |
| 未解放報酬    | 条件を満たすまで伏せる  |

---

### 9.3 図鑑ポイント

各敵には、個別の図鑑ポイントが存在する。

図鑑ポイントは、敵との戦闘や観察によって増加する。

| 行動        | ポイント増加例 |
| --------- | ------: |
| 初遭遇       |      +10 |
| 撃破        |      +5 |
| ノーダメージで撃破 |      +5 |

ポイントが満タンになると、その敵に対応したエネミーオーブを入手する。
ポイントは100がMAX
### 9.4 図鑑報酬

| 報酬      | 内容                      |
| ------- | ----------------------- |
| エネミーオーブ | オリジナルカードのエネミースロットに装着できる |
| 敵情報の詳細化 | 行動パターンや弱点が詳しく表示される      |
| 小報酬     | ゴールド、カード候補増加    |

MVPでは、図鑑報酬はエネミーオーブのみでもよい。

---

### 9.5 敵図鑑の設計意図

敵図鑑の目的は、単なる収集ではなく、敵を知ることがオリジナルカードの強化につながる体験を作ることである。

これにより、プレイヤーは以下の動機を持てる。

* まだ倒していない敵を探したい。
* 同じ敵を何度も倒して図鑑を進めたい。
* 特定のエネミーオーブを狙ってランを回したい。
* 敵の特徴を利用して、自分のカードを強化したい。
* 試練レベル攻略のために、より強いエネミーオーブを試したい。

---

## 10. カード進化システム

### 10.1 概要

一部のカードには「進化条件」が設定されている。

進化条件を持つカードは、ラン中の使用回数や条件達成によって進化する。

進化したカードは、効果量・効果内容・コスト・属性などが変化する。

---

### 10.2 基本ルール

* 進化できるカードは、カードごとに設定する。
* 進化条件は、主にラン中の使用回数とする。
* 使用回数は、そのラン中のみカウントする。
* 進化はラン中に発生する。
* 進化後のカードは、そのラン中は進化後の状態を維持する。
* ラン終了後、通常カードの進化状態はリセットされる。
* MVPでは、進化段階は1段階までとする。
* オリジナルカードの進化は、MVPでは対象外とする。

---

### 10.3 進化カードの入手

進化カードは、以下の方法で入手できる。

* カード報酬
* ショップ
* イベント
* 宝箱
* 特定の遺物効果

進化カードは通常カードより将来性があるため、出現率は高くしすぎない。

---

### 10.4 進化条件例

| 条件     | 内容                  |
| ------ | ------------------- |
| 使用回数   | このカードを5回使用すると進化     |
| 撃破条件   | このカードで敵を3体倒すと進化     |
| 防御条件   | このカードで合計30ブロック得ると進化 |
| 状態異常条件 | このカードで毒を合計10付与すると進化 |
| 代償条件   | このカードの代償を3回発動すると進化  |
| 鍛冶条件   | 鍛冶済み状態で3回使用すると進化    |

---

### 10.5 進化カード例

| 進化前         | 進化条件     | 進化後                    |
| ----------- | -------- | ---------------------- |
| 1コスト、6ダメージ  | 5回使用     | 1コスト、9ダメージ             |
| 1コスト、5ブロック  | 5回使用     | 1コスト、8ブロック             |
| 1コスト、毒2付与   | 毒を合計10付与 | 1コスト、毒3付与              |
| 2コスト、14ダメージ | 敵を2体倒す   | 2コスト、18ダメージ            |
| 1コスト、1枚ドロー  | 4回使用     | 1コスト、1枚ドロー、次のカードのコスト-1 |

---

### 10.6 進化の表示

カード詳細画面では、進化条件と進行度を表示する。

```text
進化条件：このカードを5回使用する
進行度：3 / 5
```

戦闘中は、カード上に小さな進化アイコンや進行ゲージを表示する。

スマホでは情報量が多くなりすぎないよう、詳細表示で確認できる形を基本とする。

---

### 10.7 進化カードの設計方針

進化カードは、ラン中に育てる楽しさを生むためのカードである。

ただし、進化カードを強くしすぎると、通常カードや鍛冶の価値を奪うため、以下に注意する。

* 進化前はやや控えめな性能にする。
* 進化後は通常強化済みカードより少し強い程度にする。
* 進化条件は自然に達成できるが、無条件で達成されない程度にする。
* 進化カードを取りすぎると序盤が弱くなるようにする。
* 進化カードはカード報酬に出るが、出現率は高くしすぎない。

---

## 11. 鍛冶システム

### 11.1 概要

鍛冶は、ラン中にカードや遺物を変化させるノードである。

鍛冶ノードでは、以下のどちらかを選択できる。

| 行動     | 内容                           |
| ------ | ---------------------------- |
| カード鍛冶  | 空き特殊効果スロットを持つカードに鍛冶特殊効果を付与する |
| 遺物変換 | 不要な遺物2つを捧げ、ランダムな遺物1つを得る  |

1回の鍛冶ノードで選べる行動は1つだけとする。

---

### 11.2 カード鍛冶

カード鍛冶は、空き特殊効果スロットを持つカードに、鍛冶特殊効果を付与する行動である。

休憩所が「カードの数値を伸ばす場所」だとすると、カード鍛冶は「カードの性質を変える場所」である。

カード鍛冶でできること：

* 通常カードの空き特殊効果スロットに効果を付ける。
* 進化カードの空き特殊効果スロットに効果を付ける。
* スロット付きカードの役割を変える。
* デッキの弱点を補う。

カード鍛冶でできないこと：

* エネミースロットを埋める。
* エネミーオーブを作る。
* オリジナルカードのエネミーオーブを付け替える。
* オリジナルカードを複製する。
* 遺物を強化する。

---


### 11.3 カード鍛冶の対象

カード鍛冶で選択できるカードは、以下の条件をすべて満たすカードである。

* デッキ内に存在する。
* 特殊効果スロットを持っている。
* そのスロットが空いている。
* 鍛冶対象外カードではない。

以下のカードは、原則としてカード鍛冶対象外とする。

* スロットを持たないカード
* すでにスロットが埋まっているカード
* 状態カード
* 呪いカード
* 一時生成カード
* 鍛冶対象外に設定された特殊カード

---

### 11.4 カード鍛冶の流れ

```text
空きスロットを持つカードを選ぶ
 ↓
鍛冶特殊効果の候補が提示される
 ↓
候補から1つ選ぶ
 ↓
選んだ効果がカードのスロットに入る
 ↓
カードが鍛冶済み状態になる
```

鍛冶で提示される候補は、基本的に3つとする。
候補内容は、選んだカードの種類やコストに応じて変化する。

---

### 11.5 鍛冶特殊効果

| 鍛冶特殊効果 | 内容                  | 主な対象          |
| ------ | ------------------- | ------------- |
| 毒付与    | ダメージ後、毒を付与する        | 攻撃カード         |
| 裂傷付与   | ダメージ後、裂傷を付与する       | 攻撃カード         |
| 小回復    | 使用後、HPを少量回復する       | 防御カード、技能カード   |
| 追加防御   | 使用後、少量のブロックを得る      | 防御カード、技能カード   |
| 追加ドロー  | 使用後、カードを1枚引く        | 技能カード         |
| 保持     | ターン終了時に手札に残る        | 防御カード、技能カード   |
| 反撃     | ブロック獲得後、敵に小ダメージを与える | 防御カード         |
| 除外強化   | 使用後除外される代わりに効果量が上がる | 攻撃カード、技能カード   |
| 軽量化    | コストを1下げるが、使用後除外を付ける | 高コストカード       |
| 拡散     | 単体効果の一部を敵全体に適用する    | 攻撃カード、状態付与カード |

---

### 11.6 鍛冶特殊効果の出現ルール

| カード種類   | 出やすい鍛冶特殊効果           |
| ------- | -------------------- |
| 攻撃カード   | 毒付与、裂傷付与、除外強化、拡散     |
| 防御カード   | 小回復、追加防御、保持、反撃       |
| 技能カード   | 追加ドロー、保持、軽量化         |
| 状態付与カード | 拡散、追加ドロー、保持          |
| 進化カード   | そのカードの進化方向と相性のよい効果   |
| 常駐カード   | 原則として鍛冶対象外、または専用効果のみ |

強力な組み合わせを防ぐため、以下の制限を設ける。

* 0コストカードには追加ドローが出にくい。
* 0コストカードには軽量化が出ない。
* 低コストカードには保持が出にくい。
* ドロー効果を持つカードには追加ドローが出にくい。
* エナジー回復効果を持つカードには追加ドローが出にくい。
* 進化済みカードには強すぎる鍛冶効果を出しにくい。
* すでに強力なカードには、代償付きの効果が出やすい。

---

### 11.7 カード鍛冶の制限

* 1つの特殊効果スロットには、鍛冶特殊効果を1つだけ付与できる。
* スロットが埋まったカードは再鍛冶できない。
* 鍛冶済み効果を別効果に付け替えることは原則できない。
* 鍛冶特殊効果はそのラン中のみ有効。
* ラン終了後、通常カードに付与された鍛冶特殊効果は失われる。
* 鍛冶はカードを単純に強くするのではなく、使い方を変える強化を重視する。

---

### 11.8 遺物変換

遺物変換は、不要な遺物2つを捧げて、ランダムな遺物1つを得る鍛冶アクションである。

プレイヤーは、所持している遺物から2つを選択する。

選択した遺物は失われ、代わりに新しい遺物を1つ入手する。

---

### 11.9 遺物変換ルール

| 捧げる遺物    | 入手遺物               |
| ---------- | -------------------- |
| 同じレアリティ2つ  | 1つ上のレアリティのランダム遺物   |
| 異なるレアリティ2つ | 高い方と同じレアリティのランダム遺物 |

例：

| 捧げる遺物A | 捧げる遺物B | 入手遺物           |
| -------- | -------- | ---------------- |
| 通常       | 通常       | 希少遺物           |
| 希少       | 希少       | 伝説遺物           |
| 通常       | 希少       | 希少遺物           |
| 希少       | 伝説       | 伝説遺物           |
| 伝説       | 伝説       | 伝説遺物、または専用上限処理 |

---

### 11.10 レアリティ上限

最高レアリティ同士を捧げた場合、1つ上のレアリティが存在しないため、以下のいずれかを採用する。

| 処理        | 内容                     |
| --------- | ---------------------- |
| 同レアリティ再抽選 | 最高レアリティの遺物をランダムで1つ得る |
| ボス級抽選     | 低確率でボス級遺物を得る         |
| 変換不可      | 最高レアリティ2つは変換できない       |

MVPでは、同レアリティ再抽選を推奨する。

---

### 11.11 遺物変換の制限

* 1回の鍛冶ノードで選べる行動は1つだけ。
* カード鍛冶と遺物変換を同時には行えない。
* 初期遺物は遺物変換の素材にできない。
* 呪い系遺物やイベント固定遺物は変換対象外にできる。
* 変換で得られる遺物は、すでに所持しているものを除外してもよい。
* 変換結果はランダムとする。
* 変換前に、入手遺物のレアリティだけ表示する。
* 変換後に、具体的な遺物が判明する。

---

### 11.12 鍛冶のゲーム上の役割

鍛冶の狙いは、カード報酬・遺物選択・ルート選択に新しい判断を生むことである。

プレイヤーは鍛冶ノードで以下の判断を行う。

* カードを鍛冶してデッキを強くする。
* 不要な遺物2つを変換して上位遺物を狙う。
* 鍛冶ノードまでスロット付きカードを温存する。
* 低レア遺物を温存して変換素材にする。
* 今のデッキに必要なのはカード強化か、遺物再抽選かを判断する。

---

## 12. マップ仕様

### 12.1 階層構造

1ランは複数階層で構成する。

| 階層   | 内容                |
| ---- | ----------------- |
| 第1階層 | 基礎戦闘中心。低難度        |
| 第2階層 | 敵ギミック増加。中難度       |
| 第3階層 | 高火力・複合ギミック。高難度    |
| 最終階層 | 条件達成時のみ到達可能な追加ボス戦 |

---

### 12.2 ノード種類

| ノード   | 内容              |
| ----- | --------------- |
| 通常戦闘  | 通常敵との戦闘         |
| エリート戦 | 高難度戦闘。強力な報酬あり   |
| ボス戦   | 階層最後の戦闘         |
| イベント  | 選択肢による効果        |
| 休憩    | 回復または通常強化       |
| 鍛冶    | カード鍛冶、または遺物変換 |
| ショップ  | カード・遺物・カード削除    |
| 宝箱    | 遺物や特殊報酬を獲得      |

---

### 12.3 マップ生成ルール

* 各階層は縦方向に複数段のノードを持つ。
* プレイヤーは現在ノードから接続された次ノードを選ぶ。
* ボスノードは階層最終段に1つ配置する。
* 序盤にエリート戦が連続しすぎないよう制限する。
* ボス前には休憩または鍛冶のどちらかを最低1つ配置する。
* ショップは一定確率で配置する。
* 鍛冶は休憩よりやや低確率で出現する。
* スロット付きカードや不要な遺物を持っている場合、鍛冶ノードの価値が上がるようにする。

---

## 13. 戦闘システム

### 13.1 戦闘開始処理

戦闘開始時、以下を実行する。

1. プレイヤーのデッキを山札にコピーする。
2. 山札をシャッフルする。
3. 捨て札、除外札、手札を空にする。
4. 戦闘開始時効果を発動する。
5. 初期手札を引く。
6. 敵の初期行動予定を決定する。
7. プレイヤーターンを開始する。

---

### 13.2 ターン構成

```text
プレイヤーターン開始
 ↓
ブロック初期化
 ↓
ターン開始時効果
 ↓
エナジー回復
 ↓
カードドロー
 ↓
ポーション使用
 ↓
カード使用
 ↓
カード使用回数・進化条件チェック
 ↓
ターン終了
 ↓
手札破棄
 ↓
敵ターン開始
 ↓
敵行動
 ↓
敵ターン終了
 ↓
次ターンへ
```

---

### 13.3 プレイヤーターン開始処理

| 処理      | 説明                   |
| ------- | -------------------- |
| ブロック初期化 | 原則として前ターンのブロックを0にする  |
| 状態処理    | 毒、再生、弱体などのターン開始効果を処理 |
| 遺物処理    | ターン開始時発動遺物を処理        |
| エナジー回復  | 最大エナジー分まで回復          |
| カードドロー  | 基本ドロー枚数分、山札から引く      |

---

### 13.4 カード使用処理

```text
使用可能チェック
 ↓
コスト支払い
 ↓
対象選択
 ↓
カード効果解決
 ↓
使用後効果発動
 ↓
進化進行度更新
 ↓
カードを移動
```

使用可能条件：

* 必要エナジー以上のエナジーを持っている。
* 使用条件を満たしている。
* 対象が存在する。
* 使用不可状態になっていない。

---

### 13.5 カード使用後の移動先

| カード状態   | 移動先          |
| ------- | ------------ |
| 通常カード   | 捨て札          |
| 除外属性カード | 除外札          |
| 消滅指定カード | 戦闘から完全削除     |
| 保持属性カード | ターン終了時も手札に残る |
| 一時生成カード | 戦闘終了時に消える    |

## 14. ダメージ・防御仕様

### 14.1 ダメージ計算

最終ダメージは、基礎ダメージに各種補正をかけた後、ブロックで軽減する。

処理順：

1. 基礎ダメージを取得
2. 攻撃側の強化・弱体を適用
3. 防御側の被ダメージ増減を適用
4. ブロックで軽減
5. HPに反映
6. 被弾時効果を発動
7. 撃破判定

---

### 14.2 ブロック

* ブロックはダメージを肩代わりする一時防御値。
* 通常はプレイヤーターン開始時に0になる。
* 一部効果により次ターンへ持ち越せる。
* ブロックを超えたダメージのみHPに入る。

---

### 15.2 カード種類

| 種類       | 主な役割                 |
| -------- | -------------------- |
| 攻撃       | 敵にダメージを与える           |
| 防御       | ブロックや耐久効果を得る         |
| 技能       | ドロー、エナジー操作、状態付与など    |
| 常駐       | 戦闘中、持続効果を与える         |

| 状態カード    | デッキに混入する妨害カード        |
| 呪いカード    | ラン中残りやすいデメリットカード     |
| オリジナルカード | プレイヤーが作成し、持ち込む特別なカード |

---

### 15.3 カード属性

| 属性   | 説明                |
| ---- | ----------------- |
| 除外   | 使用後、その戦闘中は再使用不可   |
| 保持   | ターン終了時に捨てられない     |
| 一時生成 | 戦闘終了時に消える         |
| 廃棄   | 使用後、完全に消える        |
| 自動発動 | 引いた時や条件達成時に発動     |
| 使用不可 | 手札にあっても使えない       |
| 反復不可 | 1ターンに1回しか使用できない   |
| 不安定  | 使用時にランダムな副作用が発生する |
| 進化可能 | 条件達成で進化する         |

---

### 15.4 特殊効果スロット

一部の通常カード・進化カードには、特殊効果スロットが存在する。

特殊効果スロットは、鍛冶ノードで鍛冶特殊効果を付与するための枠である。

| 状態       | 内容              |
| -------- | --------------- |
| スロットなし   | 鍛冶対象にならない       |
| 空きスロットあり | 鍛冶で特殊効果を付与できる   |
| 使用済みスロット | すでに鍛冶特殊効果が付いている |

---

### 15.5 スロット付きカードの役割

スロット付きカードは、入手時点ではやや控えめな性能でもよい。
その代わり、鍛冶によって強化できる将来性を持つ。

| カード例    | 初期性能        | スロット |
| ------- | ----------- | ---- |
| 軽い防御カード | 1コスト、5ブロック  | 空き1  |
| 小攻撃カード  | 1コスト、6ダメージ  | 空き1  |
| 補助カード   | 1コスト、1枚ドロー  | 空き1  |
| 重い攻撃カード | 2コスト、14ダメージ | 空き1  |

MVPでは、カード1枚につき特殊効果スロットは最大1つまでとする。

---

### 15.6 内部相性タグ

カードには、報酬抽選やバランス調整に使用する内部タグを設定する。

内部タグは、プレイヤーには原則として表示しない。

| 内部タグ | 内容                      |
| ---- | ----------------------- |
| 攻撃   | ダメージを与えるカード             |
| 防御   | ブロックや耐久に関係するカード         |
| 連撃   | 複数回使用、低コスト、手数に関係するカード   |
| 毒    | 毒を付与・強化するカード            |
| 裂傷   | 被ダメージ増加や弱体に関係するカード      |
| 弱体   | 敵の攻撃力低下や被ダメージ増加に関係するカード |
| ドロー  | カードを引くカード               |
| 回復   | HPを回復するカード              |
| 鍛冶   | スロットや鍛冶と相性が良いカード        |
| 進化   | 進化条件を持つカード              |
| 全体   | 敵全体に影響するカード             |

---

### 15.7 内部相性タグの表示方針

内部相性タグは、プレイヤーに直接表示しない。

表示しない理由は、プレイヤーの選択を誘導しすぎないためである。

タグを表示すると、プレイヤーは「相性タグが付いているカードを取るのが正解」と感じやすくなる。
その結果、カード選択の自由度が下がり、毎回似た構築になりやすい。

---

### 15.8 通常強化

カードは休憩所、イベント、報酬などで通常強化できる。

| 強化内容    | 例             |
| ------- | ------------- |
| ダメージ増加  | 6ダメージ → 9ダメージ |
| ブロック増加  | 5ブロック → 8ブロック |
| コスト減少   | 2コスト → 1コスト   |
| 追加効果    | ダメージ後に1枚ドロー   |
| デメリット軽減 | HP消費3 → HP消費2 |

原則として、1枚のカードは1回のみ通常強化可能とする。

---

### 15.9 鍛冶強化

鍛冶強化は、鍛冶ノードで空き特殊効果スロットに鍛冶特殊効果を付与することで行われる。

| 鍛冶強化 | 例                |
| ---- | ---------------- |
| 毒付与  | ダメージ後、毒を付与       |
| 小回復  | 使用後、HPを少量回復      |
| 反撃   | ブロック獲得後、小ダメージ    |
| 保持化  | ターン終了時に手札に残る     |
| 全体化  | 単体対象を敵全体にするが数値低下 |
| 軽量化  | コストを下げるが代償を追加    |

鍛冶強化は、空きスロットを持つカードにのみ付与できる。

---

## 16. デッキ・山札・捨て札仕様

### 16.1 ドロー処理

カードを引く時、山札が不足した場合は以下を行う。

1. 捨て札をシャッフルする。
2. 捨て札を山札に移動する。
3. 必要枚数までカードを引く。
4. 捨て札が空の場合は引けない。

---

### 16.2 ターン終了時処理

ターン終了時、以下を実行する。

* 使用されなかった通常手札を捨て札へ送る。
* 保持属性のカードは手札に残す。
* ターン終了時効果を発動する。
* エナジーは原則として次ターンに持ち越さない。

---

## 18. 遺物仕様

### 18.1 遺物の役割

遺物はラン中に獲得できる常時効果アイテム。
カードと異なり、基本的にデッキには入らない。

遺物は、オリジナルカードの方向性を補強したり、通常カードや鍛冶済みカードの価値を高めたりする役割を持つ。

---

### 18.2 初期遺物

初期遺物は、開始デッキごとに設定される遺物である。

初期遺物は、開始デッキの方向性を補強するが、それだけで勝てるほど強くしない。

| 開始デッキ | 初期遺物   | 効果                        |
| ----- | ------ | ------------------------- |
| 均衡型   | 古びた紋章  | 戦闘開始時、少量のブロックを得る          |
| 連撃型   | 小さな歯車  | 1ターンに3枚目のカードを使うと小ダメージ     |
| 守護型   | ひび割れた盾 | ターン終了時にブロックが残っていれば小ダメージ   |
| 侵蝕型   | 黒い小瓶   | 1ターンに初めて状態異常を付与した時、その数値+1 |

初期遺物は遺物変換の素材にできない。

---

### 18.3 遺物効果例

| タイプ        | 効果例                 |
| ---------- | ------------------- |
| 戦闘開始型      | 戦闘開始時にブロックを得る       |
| ターン開始型     | 毎ターン追加でカードを1枚引く     |
| 条件達成型      | 3枚目に使った攻撃カードのダメージ増加 |
| 報酬強化型      | 獲得ゴールド増加            |
| 進化連動型      | 進化カードの進行度が増えやすくなる   |
| 図鑑連動型      | 図鑑ポイント獲得量が少し増える     |
| リスク型       | 最大HP減少の代わりに火力上昇     |
| エネミーオーブ連動型 | 装着中のエネミーオーブ効果を強化する  |
| 鍛冶連動型      | 鍛冶済みカードを使うと追加効果     |
| スロット連動型    | 空きスロット付きカードの出現率が上がる |

---

## 19. ポーション仕様

### 19.1 概要

ポーションは、ラン中に獲得・使用できる消耗品アイテムである。

カードや遺物と異なり、使用すると消費される。

ポーションは、戦闘中の一時的な打開策として機能する。

---

### 19.2 基本ルール

* ポーションはラン中のみ使用できる。
* 使用すると消費される。
* 所持数には上限を設ける。
* 初期所持数は0とする。
* MVPでは、ポーション所持上限は2個とする。
* ポーションは主に戦闘報酬、ショップ、イベント、宝箱から入手する。
* 一部ポーションは戦闘中のみ使用できる。
* 一部ポーションは非戦闘時にも使用できる。
* ラン終了時、未使用ポーションは失われる。

---

### 19.3 ポーション所持枠

| 項目 | 内容 |
|---|---|
| 初期所持数 | 0 |
| 所持上限 | 2 |
| 上限拡張 | 遺物やイベントで拡張可能 |
| ラン終了時 | すべて消滅 |

---

### 19.4 ポーション例

| ポーション | 効果 |
|---|---|
| 小回復ポーション | HPを少量回復する |
| 攻撃ポーション | 敵1体にダメージを与える |
| 防御ポーション | ブロックを得る |
| ドローポーション | カードを2枚引く |
| エナジーポーション | このターン、エナジーを1得る |
| 解毒ポーション | 毒など一部の弱体を解除する |
| 鍛冶火花 | この戦闘中、カード1枚の効果量を一時的に増やす |
| 図鑑ポーション | 戦闘後、その敵の図鑑ポイントを少し増やす |

---

### 19.5 ポーションの入手方法

| 入手方法 | 内容 |
|---|---|
| 戦闘報酬 | 稀にポーションが提示される |
| ショップ | ゴールドで購入できる |
| イベント | 選択肢によって獲得できる |
| 宝箱 | 遺物の代わり、または追加報酬として出現する |
| 遺物効果 | 一部遺物によって生成される |

---

### 19.6 ポーションの設計方針

ポーションは、プレイヤーの判断で危機を切り抜けるための要素である。

ただし、強すぎるポーションは戦闘バランスを壊すため、以下に注意する。

* ボスを一撃で倒せる効果は避ける。
* 無限ドローや無限エナジーにつながる効果は避ける。
* 所持上限を低めにする。
* 入手頻度を高くしすぎない。
* 強力なポーションはショップ価格を高くする。
* エリート戦やボス戦で使うか、通常戦で温存するかの判断を生む。

---

## 20. 敵仕様

### 20.1 敵の役割

敵は、単にプレイヤーにダメージを与える存在ではなく、プレイヤーのデッキ方針を試す存在として設計する。

| 敵タイプ | 試す内容               |
| ---- | ------------------ |
| 高HP敵 | 継続火力があるか           |
| 高火力敵 | 防御手段があるか           |
| 複数敵  | 単体特化に偏っていないか       |
| 防御敵  | 防御突破手段があるか         |
| 弱体敵  | デバフ対策があるか          |
| 召喚敵  | 処理速度があるか           |
| 反撃敵  | 多段攻撃に頼りすぎていないか     |
| 妨害敵  | ドローやコンボに依存しすぎていないか |

---

### 20.2 敵行動予定

敵は次の行動を事前に表示する。

| 表示 | 意味              |
| -- | --------------- |
| 攻撃 | 次ターン攻撃する        |
| 防御 | ブロックを得る         |
| 強化 | バフを付与する         |
| 弱体 | プレイヤーにデバフを与える   |
| 召喚 | 新しい敵を出現させる      |
| 特殊 | 条件付き行動を行う       |
| 不明 | 複合行動または初見用の特殊行動 |

---

### 20.3 ボス設計

ボスは、各階層でプレイヤーのデッキ完成度を試す。

| 階層   | ボスの役割             |
| ---- | ----------------- |
| 第1階層 | 基本的な攻防ができているか確認する |
| 第2階層 | 状態異常や複数敵への対応を試す   |
| 第3階層 | デッキの弱点を突き、完成度を試す  |
| 最終階層 | オリジナルカードへの依存度を試す  |

最終ボスには、オリジナルカードへの依存度を試すギミックを持たせると独自性が出る。

例：

* 同じカードを使い続けると敵が耐性を得る。
* 強力なカードを使うほどボスが強化される。
* 代償付きカードを使うと追加報酬条件を満たせる。
* オリジナルカードを使いにくいターンがある。
* オリジナルカード以外のデッキ完成度も要求される。

---

## 21. 報酬仕様

### 21.1 戦闘報酬

通常戦闘後、以下から報酬を生成する。

* ゴールド
* カード選択報酬
* 稀に遺物
* 稀にポーション
* 稀に消費アイテム
* 敵図鑑ポイント

カード・遺物・ポーションは、解放済みのものだけが報酬候補に含まれる。

---

### 21.2 カード報酬

カード報酬では、複数枚の候補から1枚を選択する。
選ばない選択肢も用意する。

カード候補生成ルール：

* レアリティごとに重み付けする。
* 階層や所持遺物に応じて補正する。
* オリジナルカードの効果傾向に少しだけ寄せた候補を出す。
* 開始デッキの内部相性タグに少しだけ寄せた候補を出す。
* 進化カードは一定確率で出現する。
* スロット付きカードは一定確率で出現する。
* 解放済みカードのみ候補に含める。
* ただし寄せすぎると毎回同じデッキになるため、完全な誘導はしない。

---

### 21.3 報酬3枠の役割

カード報酬は、以下のような役割で候補を生成する。

| 報酬枠 | 役割 |
|---|---|
| 枠A | 開始デッキと相性が良いカードが少し出やすい |
| 枠B | 防御、ドロー、弱点補助などの汎用カード |
| 枠C | ランダム、進化カード、または別方向に伸ばせるカード |

---

### 21.4 報酬補正の注意点

* 相性タグを持つカードを確定で出してはいけない。
* 報酬3枠すべてを開始デッキ寄りにしない。
* 相性カードは出やすくするだけで、必ず出るわけではない。
* 弱点補助や方向転換のカードも必ず残す。
* 報酬補正は、当たりを増やすより外れを減らす目的で使う。
* 報酬画面で「おすすめ」「相性あり」「このデッキ向け」などの表示はしない。

---

### 21.5 報酬の方向性

| 報酬タイプ | 内容 |
|---|---|
| 伸ばす報酬 | オリジナルカードや開始デッキの長所をさらに強くする |
| 補う報酬 | デッキの弱点を補う |
| 変える報酬 | 当初の方針から別戦略へ転換できる |
| 育てる報酬 | 空きスロットを持ち、鍛冶によって強化できるカード |
| 進化報酬 | ラン中に育てることで強くなるカード |
| 消耗品報酬 | ポーションによって一時的な打開策を得る |

---

### 21.6 ポーション報酬

ポーションは戦闘報酬、ショップ、イベント、宝箱で入手できる。

ポーション報酬は、カードや遺物ほど高頻度に出しすぎない。

ポーション報酬の役割は、次の戦闘やボス戦に向けた一時的な備えを与えることである。

---

### 21.7 ボス報酬

ボス撃破後は、通常より強力な報酬を提示する。

例：

* 高レア遺物
* 最大エナジー増加系遺物
* 大量ゴールド
* 進化カード
* 特殊カード
* オリジナルカードの代償軽減
* 敵図鑑ポイント増加
* 解放条件の進行
* 次階層への進行

---

## 22. ショップ仕様

ショップでは以下の行動が可能。

| 項目 | 内容 |
|---|---|
| カード購入 | 表示カードをゴールドで購入 |
| 遺物購入 | 表示遺物をゴールドで購入 |
| ポーション購入 | 表示ポーションをゴールドで購入 |
| カード削除 | デッキからカードを1枚削除 |
| オリジナルカード調整 | ラン中限定で軽微な調整を行う |

ショップに出現するカード・遺物・ポーションは、解放済みのものから抽選される。

---

### 22.1 オリジナルカード調整

ショップでは、オリジナルカードを根本から作り直すことはできない。
ただし、軽微な調整は可能とする。

例：

* 代償を少し軽くする。
* 効果量を少し下げてコストを下げる。
* 不安定効果を別の代償に変える。
* 使用条件を少し緩くする。

ラン中に自由編集できると戦略が壊れるため、調整は高額かつ限定的にする。

---

### 22.2 ポーション購入

ショップでは、ポーションを購入できる。

ポーションは所持上限を超えて購入できない。

強力なポーションは価格を高めに設定し、購入するかカード・遺物にゴールドを使うかの判断を生む。

---

## 23. 休憩所仕様

休憩所では、基本的に以下のどちらかを選ぶ。

| 行動 | 内容             |
| -- | -------------- |
| 休憩 | 最大HPの一定割合を回復   |
| 強化 | デッキ内カードを1枚通常強化 |

遺物やイベントにより、追加選択肢を開放できる。

例：

* カード削除
* 最大HP増加
* ランダムカード変化
* 一時バフ獲得
* オリジナルカードの代償軽減

---

## 24. イベント仕様

### 24.1 イベントの役割

イベントは、戦闘以外でデッキやラン方針を変化させる場所である。

特に本作では、オリジナルカード・エネミーオーブ・進化カード・スロット付きカードに関わるイベントを用意することで、独自性を強める。

---

### 24.2 イベント例

| イベント    | 選択肢例                    |
| ------- | ----------------------- |
| 古い標本    | 敵図鑑ポイントを得るが、呪いを得る       |
| 暴走する炉   | カード1枚を大幅強化するが、最大HPを失う   |
| 忘れられた商人 | オリジナルカードの代償を別の代償に変える    |
| 錆びた祭壇   | HPを払って遺物を得る             |
| 空白の刻印   | カード1枚に空き特殊効果スロットを追加する   |
| 進化の火種   | 進化カード1枚の進行度を増やす         |
| 禁じられた改稿 | オリジナルカードを強化するが、追加の代償を得る |

オリジナルカードの複製や一時生成を行うイベントは、MVPでは実装しない。

---

## 25. 勝敗条件

### 25.1 戦闘勝利条件

戦闘中のすべての敵を倒す。

---

### 25.2 戦闘敗北条件

プレイヤーのHPが0以下になる。

---

### 25.3 ランクリア条件

最終階層のボスを撃破する。

---

### 25.4 追加クリア条件

条件を満たしている場合、真ボス戦へ進む。

真ボス到達条件例：

* 特定のイベントを達成している。
* 代償付きのオリジナルカードで一定階層を突破している。
* 特定の鍛冶強化を行っている。
* 特定の遺物を所持している。
* 特定の敵図鑑報酬を解放している。

---

## 26. 試練レベル

### 26.1 概要

試練レベルは、ランクリア後に解放される高難度やりこみ要素である。

通常難度をクリアすると試練レベル1が解放され、試練レベル1をクリアすると試練レベル2が解放される。

試練レベルが上がるほど、敵・報酬・マップ・鍛冶・オリジナルカード運用に不利な条件が追加される。

---

### 26.2 基本ルール

* 試練レベルは最大20程度を想定する。
* 試練レベルは開始デッキごとに進行を管理する。
* 各レベルは、前レベルまでの不利条件を引き継ぐ。
* 試練レベルが上がっても、単純な数値増加だけにしない。
* 高レベルでは、デッキ構築やルート選択の判断がより重要になる。

---

### 26.3 開始デッキごとの進行

| 開始デッキ | 到達試練レベル |
| ----- | ------: |
| 均衡型   |       8 |
| 連撃型   |       3 |
| 守護型   |       5 |
| 侵蝕型   |       0 |

これにより、プレイヤーは複数の開始デッキを使い込む理由を持てる。

---

### 26.4 試練レベル案

| レベル | 追加される不利条件           |
| --: | ------------------- |
|   1 | 敵HPが少し増える           |
|   2 | 通常敵の攻撃力が少し増える       |
|   3 | 初期ゴールドが減る           |
|   4 | 休憩の回復量が減る           |
|   5 | エリート敵が強化される         |
|   6 | カード報酬のスキップ報酬が減る     |
|   7 | ショップ価格が上がる          |
|   8 | ボスのHPが増える           |
|   9 | 状態異常を使う敵が増える        |
|  10 | ラン開始時に軽い呪いカードを1枚得る  |
|  11 | 鍛冶ノードの出現率が少し下がる     |
|  12 | スロット付きカードの出現率が少し下がる |
|  13 | 敵が初期ブロックを持つことがある    |
|  14 | 一部イベントのリスクが増える      |
|  15 | 山札を1周するたびに敵が少し強化される |
|  16 | ボスがオリジナルカード使用に反応する  |
|  17 | 代償付きカードの代償が少し重くなる   |
|  18 | 最終階層の通常敵が強化される      |
|  19 | 休憩・鍛冶の選択が一部制限される    |
|  20 | 最終ボスが専用強化を得る        |

---

### 26.5 試練レベルの設計方針

試練レベルは、単純に敵HPや攻撃力を増やすだけではなく、プレイヤーの判断を難しくすることを目的とする。

高難度では、以下のような要素を少しずつ厳しくする。

* 回復しづらい。
* 鍛冶しづらい。
* ショップで解決しづらい。
* デッキの弱点を突かれやすい。
* オリジナルカードに依存しすぎると咎められる。
* スロット付きカードや進化カードを育てる余裕が減る。

---

## 27. 実績・称号・進行度解放システム

### 27.1 概要

実績・称号・進行度解放は、ランをまたいでプレイヤーの達成状況を記録し、新しい目標や報酬を与える恒久的なやりこみ要素である。

これらは、プレイヤーを単純に強くするためではなく、プレイの幅を広げ、次のランで試したい選択肢を増やすために使う。

---

### 27.2 実績システム

実績は、プレイヤーが特定条件を達成した時に解放される恒久的な記録である。

実績は、やりこみやプレイ方針の提示として機能する。

一部の実績は、カード・遺物・ポーション・称号などの解放条件になる。

基本ルール：

* 実績はセーブデータ単位で管理する。
* 実績はランをまたいで保存される。
* 実績達成時、通知を表示する。
* 実績には報酬あり・報酬なしの両方を用意する。
* 実績の一部は称号やカード、遺物、ポーションの解放条件になる。
* 実績はゲームプレイを強制しすぎないようにする。

---

### 27.3 実績例

| 実績 | 条件 | 報酬 |
|---|---|---|
| 初めての一歩 | 初めてランを開始する | なし |
| 初勝利 | 初めてボスを撃破する | カード解放 |
| 鍛冶職人 | 鍛冶を累計10回行う | 称号 |
| 図鑑研究者 | 敵図鑑を10体分進める | 遺物解放 |
| 毒使い | 毒で敵を累計50体倒す | カード解放 |
| 守りの達人 | 1戦闘で合計100ブロック得る | 称号 |
| 進化の導き手 | 進化カードを累計20回進化させる | 遺物解放 |
| 変換好き | 遺物変換を累計10回行う | 称号 |
| 試練を越える者 | 試練レベル5をクリアする | カード解放 |
| 深層到達者 | 試練レベル10をクリアする | 遺物解放 |

---

### 27.4 称号システム

称号は、実績や特定条件の達成によって獲得できる表示用のコレクション要素である。

称号は基本的にゲームバランスへ直接影響しない。

プレイヤーの達成状況や遊び方を表現するための要素として扱う。

基本ルール：

* 称号は実績、敵図鑑、試練レベル、特殊なプレイ条件などで獲得できる。
* 称号はプロフィール画面やリザルト画面に表示できる。
* 称号は基本的に能力補正を持たない。
* 称号に能力効果を持たせる場合は、別システムとして慎重に扱う。
* MVPでは、称号は表示のみでよい。

---

### 27.5 称号例

| 称号 | 獲得条件 |
|---|---|
| 初めての挑戦者 | 初めてランを開始する |
| 鍛冶見習い | 初めて鍛冶を行う |
| 炉の常連 | 鍛冶を累計20回行う |
| 図鑑蒐集家 | 敵図鑑を一定数解放する |
| 毒の理解者 | 毒系エネミーオーブを入手する |
| 守護者 | 守護型でランをクリアする |
| 連撃の使い手 | 連撃型でランをクリアする |
| 深層の踏破者 | 試練レベル10をクリアする |
| 変換師 | 遺物変換を一定回数行う |
| 進化を導く者 | 進化カードを一定数進化させる |

---

### 27.6 進行度解放システム

進行度解放は、プレイヤーの達成状況に応じて、新しいカード・遺物・ポーション・エネミーオーブ候補などを解放するシステムである。

進行度解放は、プレイヤーに長期的な目標を与える。

ただし、解放要素はプレイヤーを単純に強くするのではなく、選択肢を増やす方向で設計する。

---

### 27.7 解放対象

| 解放対象 | 内容 |
|---|---|
| カード | カード報酬やショップに出現するカードを追加 |
| 遺物 | 遺物報酬やショップに出現する遺物を追加 |
| ポーション | 報酬やショップに出現するポーションを追加 |
| エネミーオーブ | 敵図鑑から入手できるオーブを追加 |
| 開始デッキ | 将来的な追加開始デッキ |
| 称号 | 表示用コレクション |
| 試練レベル | 高難度段階 |

---

### 27.8 解放条件例

| 条件 | 解放例 |
|---|---|
| 初回クリア | 新しいカード数枚を解放 |
| 特定開始デッキでクリア | その戦術と相性のよい遺物を解放 |
| 敵図鑑を一定数埋める | 図鑑連動遺物を解放 |
| エネミーオーブを一定数入手 | オーブ連動カードを解放 |
| 進化カードを一定回数進化 | 進化連動遺物を解放 |
| 鍛冶を一定回数行う | 鍛冶関連カードを解放 |
| 遺物変換を一定回数行う | 変換関連遺物を解放 |
| 試練レベルを突破 | 高難度向けカードや遺物を解放 |

---

### 27.9 解放設計の注意点

* 解放要素は、単純な上位互換にしない。
* 初期状態でもゲームが成立するようにする。
* 解放によって選択肢が増える設計にする。
* 強すぎるカードや遺物を後半解放に集中させない。
* 解放済み要素が増えすぎて、報酬プールが薄まらないようにする。
* 必要に応じて、開始デッキごとに出現しやすさを調整する。
* 解放要素は、プレイヤーに「次のランで試したい」と思わせるものにする。

## 34. ゲームとしての設計方針

### 34.1 オリジナルカードは強すぎても弱すぎてもいけない

オリジナルカードは本作の主役である。
そのため、使っていて楽しい強さは必要。

ただし、1枚で勝てるほど強いと、デッキ構築の意味がなくなる。

理想は以下。

* 序盤では頼りになる。
* 中盤以降はサポートカードや遺物が必要になる。
* 強力な設計ほど代償が重くなる。
* エネミーオーブによって個性が出る。
* そのカードを中心にデッキを組みたくなる。
* ただし、そのカードだけでは勝ち切れない。

---

### 34.2 エネミーオーブは長期目標として機能させる

エネミーオーブは、単なる強化パーツではなく、敵図鑑を進める理由として機能させる。

プレイヤーは、敵と戦うことで図鑑を進め、図鑑を進めることでオリジナルカードを強化できる。

```text
敵と戦う
 ↓
敵図鑑が進む
 ↓
エネミーオーブを得る
 ↓
オリジナルカードに装着する
 ↓
新しい戦略でランに挑む
```

---

### 34.3 開始デッキはキャラクターの代わりに戦術差を作る

本作ではキャラクターを増やさない。
その代わり、開始デッキと初期遺物によって、ランごとの戦術差を作る。

開始デッキは以下を満たすように設計する。

* プレイ感が違う。
* 得意・苦手がある。
* 初期遺物と噛み合っている。
* オリジナルカードとの組み合わせで幅が出る。
* 報酬補正が強すぎず、毎回同じ構築にならない。

---

### 34.4 報酬補正は外れを減らすために使う

報酬補正は、プレイヤーに正解を押し付けるものではない。

目的は、開始デッキとまったく噛み合わない報酬を減らすことである。

避けるべきこと：

* 相性タグを表示する。
* おすすめカードを明示する。
* 相性カードを確定で出す。
* 報酬候補をすべて開始デッキ寄りにする。

目指すこと：

* 少しだけ噛み合いやすくする。
* 弱点補助カードも出す。
* 方向転換できるカードも出す。
* プレイヤーが自分で判断した感覚を残す。

---

### 34.5 進化カードはラン中の育成要素として扱う

進化カードは、ラン中に育てる楽しさを作るための要素である。

進化カードは、取った瞬間から強いカードではなく、使い込むことで強くなるカードとして扱う。

これにより、プレイヤーは以下の判断を行う。

* 今すぐ強いカードを取る。
* 将来強くなる進化カードを取る。
* 進化条件を満たすためにデッキを調整する。
* 進化カードを使うためにルートや報酬を選ぶ。

---

### 34.6 鍛冶はカード強化と遺物変換の選択にする

鍛冶は、単純な強化ノードではなく、ラン中の方針転換を行う場所とする。

プレイヤーは、カードを強くするか、不要な遺物を変換するかを選ぶ。

この選択によって、鍛冶ノードの価値を状況ごとに変化させる。

---

### 34.7 ポーションは短期判断を作る

ポーションは、ラン中の一時的な切り札として扱う。

ポーションは恒久的な強化ではなく、戦闘中の危機を切り抜けるための消耗品である。

プレイヤーは、以下の判断を行う。

* 今使って被害を減らす。
* エリート戦やボス戦まで温存する。
* ショップで買うか、カードや遺物にゴールドを使うか選ぶ。
* 所持枠が埋まった時、どのポーションを残すか選ぶ。

---

### 34.8 実績・称号・進行度解放は長期目標にする

実績・称号・進行度解放は、ラン単位の勝敗とは別に、長く遊ぶ理由を作るための要素である。

ただし、これらはプレイヤーを単純に強くするためではなく、選択肢を増やすために使う。

解放要素は、次のランで試したくなるカード・遺物・ポーションを中心にする。

称号は基本的に表示用とし、能力補正は持たせない。

---

### 34.9 ランごとの目標を明確にする

プレイヤーはラン開始時点で、ある程度の方針を持てる。

例：

* 開始デッキの長所を伸ばす。
* 開始デッキの弱点を補う。
* オリジナルカードを何回も使うデッキにする。
* オリジナルカードの代償を補うデッキにする。
* エネミーオーブの効果を活かす。
* 進化カードを育てる。
* 空きスロット付きカードを拾い、鍛冶で強化する。
* 不要な遺物を集めて変換する。
* ポーションを温存して難所を突破する。
* 鍛冶ノードを目指すルートを選ぶ。
* 実績や解放条件を狙ってプレイする。

### 34.10 失敗しても次のカード作成につながる

ランに負けた時、ただ失敗で終わるのではなく、
「次はカードをこう変えよう」と思えることを重視する。

リザルト画面では以下を表示する。

* 選択した開始デッキ
* オリジナルカードの使用回数
* 与えた合計ダメージ
* 防いだ合計ダメージ
* 発生した代償の回数
* エネミーオーブ効果の発動回数
* 進化したカード
* 進化できなかったカード
* 鍛冶済みカードの活躍
* 遺物変換の結果
* 使用したポーション
* 初期遺物の発動回数
* 敵図鑑の進行
* 実績の進行
* 解放された要素
* 次回の調整候補

## 35. 開発時の注意

* キャラクターではなく、開始デッキとオリジナルカードがプレイヤーの個性になるようにする。
* 開始デッキは実質的なクラスだが、キャラクターとして扱わない。
* 初期遺物は開始デッキの方針を補強するが、強くしすぎない。
* オリジナルカード作成時に特殊効果を直接付与できないようにする。
* オリジナルカードの特殊効果は、敵図鑑由来のエネミーオーブで付与する。
* エネミースロットは鍛冶で埋められない。
* エネミーオーブは、敵を知ることとカードを育てることをつなげる要素にする。
* オリジナルカードを強くしすぎず、弱くしすぎない。
* カード作成は自由度よりも、わかりやすさを優先する。
* 最初から大量の効果を作らず、攻撃・防御・ドロー・状態付与だけで試す。
* 代償は罰ではなく、戦略になるように設計する。
* 鍛冶で付与する特殊効果はラン中のみの成長要素として扱う。
* 鍛冶は、カード強化か遺物変換のどちらかを選ぶ場所にする。
* 遺物変換は強力になりすぎないよう、素材にできない遺物を設定する。
* 初期遺物は遺物変換の素材にできない。
* スロット付きカードは、カード報酬の判断を面白くするために使う。
* スロット付きカードを強くしすぎると、通常カードの価値が下がるため注意する。
* 進化カードは強くしすぎず、ラン中に育てる楽しさを重視する。
* 進化カードを取りすぎると序盤が弱くなるようにする。
* ポーションは戦闘の保険として扱い、戦略の中心にしすぎない。
* ポーション所持上限は低めにする。
* 実績はプレイヤーの遊び方を広げる目的で使う。
* 実績のために苦痛なプレイを強制しない。
* 称号は基本的に表示用とし、能力補正は持たせない。
* 進行度解放は、強さのインフレではなく選択肢の拡張として扱う。
* 解放済み要素が増えすぎて報酬プールが散らばらないようにする。
* 新しいカードや遺物を解放したら、次のランで試したくなる導線を作る。
* 内部相性タグはプレイヤーに見せない。
* 報酬補正は強すぎないようにする。
* シナジーを保証するのではなく、外れを減らす目的で使う。
* 敵図鑑は単なる収集要素ではなく、オリジナルカード強化の導線にする。
* 敵はプレイヤーの偏った構築を試すように設計する。
* 試練レベルは単純な数値上昇だけにしない。
* 高難度では、鍛冶・回復・ショップ・報酬選択の判断を難しくする。
* ラン終了後に「次は別のカードを作りたい」と思える導線を作る。
* スマホではカード作成画面を複雑にしすぎない。
* スマホではスロット状態や進化状態を文字ではなくアイコンで表示する。
* MVPでは、オリジナルカード1枚持ち込みに絞る。
